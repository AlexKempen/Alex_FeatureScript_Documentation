Edge derive¶

Edge derive makes it possible to quickly derive length-specific parts into a part studio. Mate connectors attatched to derived parts are automatically converted into selectable points, and various positioning options are also avaialable by default.

The Edge derive FeatureScript can be found here: Edge derive document

Steps for deriving entities¶

  1. Create a Edge derive feature by selecting it from your FeatureScript dropdown.

  2. Specify Entities to import.

  3. Select one or more edges to derive parts along.

  4. For each selected edge, a set of manipulators are automatically created in the graphics window which enable positioning.

  1. Choose whether to Move parts relative to their reference edges.

  2. Choose whether to Delete planes and sketches from Entities to import.

  3. Click .

Steps for setting up parts to be derived with edge derive¶

  1. Create a new part studio.

  2. In the Configuration panel on the right side of the screen, add a Confirguation variable named “length” (lowercase ‘l’).

  3. Use the configuration to drive the length of your part.

  4. If desired, add one or more mate connectors with their z-axis facing upwards to the middle of the part. When the part is derived using Edge derive, these mate connectors will be converted into selectable points.

  5. Add additional configurations and modifications to the part studio as desired.
